2. 템플릿(Templates)이란?

  • Django에서 사용자에게 보여줄 화면을 구성하는 HTML 역할
  • 보통 templates 폴더 안에 HTML 파일을 생성
  • views.py에서 데이터를 전달받아 HTML을 렌더링함
  • Django 템플릿 언어(DTL)를 사용하여 동적인 화면 구성 가능

3. 템플릿 사용해보기

  1. HTML, CSS 파일 작성
  2. views.py 작성
  3. urls.py 작성
  4. HTML 페이지와 view 연결

4. 템플릿 언어(Django Template Language, DTL)

  • HTML 안에서 동적 데이터 처리 가능
  • 파이썬 문법을 기반으로 동작
  • 제공 기능: 변수 출력, 조건문, 반복문 등

템플릿 변수

문법 설명
{{ 변수명 }} 변수 출력
{{ 변수명.속성명 }} 객체의 속성 출력
{{ list명.순서 }} 리스트 인덱스 출력
{{ dict.key }} 딕셔너리 키 값 출력

템플릿 태그

문법 설명
{% if 조건 %}
{% else %}
{% endif %} 		조건문
{% for item in list %}
{{ item }}
{% endfor %} 		반복문

5. 템플릿 상속이란?

공통된 HTML 구조를 "부모 템플릿"으로 정의하고,
이를 "자식 템플릿"에서 확장하여 사용하는 방식이다.

예를 들어, 여러 HTML 파일에서 동일하게 반복되는 head, nav bar 등은 하나의 부모 템플릿으로 만들고,
자식 템플릿에서 필요한 부분만 따로 작성하여 유지 보수와 재사용성을 높일 수 있다.

  • {% block content %} : 블록 범위 시작
  • {% endblock %} : 블록 범위 끝
  • {% extends 'base.html' %} : base.html을 확장하겠다는 의미

6. static이란?

웹 페이지에서 사용하는 CSS, 이미지 등 정적인 파일을 저장하는 디렉토리이다.

static 파일을 HTML에서 사용하기 위해서는 다음과 같은 문법을 사용한다.

  • {% load static %} : HTML 상단에 선언
  • <img src="{% static 'images/파일명.확장자' %}" alt=""> : 이미지 파일 사용
  • <link rel="stylesheet" type="text/css" href="{% static 'css/파일명.css' %}"> : CSS 포함

또한 settings.py 파일에서 static 경로를 다음과 같이 설정해줘야 한다.

  • STATICFILES_DIRS = [os.path.join(BASE_DIR, 'project', 'static')]
  • STATIC_ROOT = os.path.join(BASE_DIR, 'static')
